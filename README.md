# dubtrackmobile

## Install node/react-native cli first
```brew install node```  
```brew install watchman```  
```npm install -g react-native-cli```  

## Installing the project
```git clone https://github.com/chrissbendel/dubtrackmobile```  
```cd dubtrackmobile```  
```npm install```  
#####Finally  
```react-native run-ios```
